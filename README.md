# moviefindr API

An Express API that fetches data from The Movie DataBase(TMDB)

# Stack Used
* NodeJs
* Express
* Jest
* MongoDB

## Project Setup

Clone Project

```git clone https://github.com/gathoni-k/moviefindr-api```

Install dependencies

```yarn install```

 ## Scripts
 
 Start
 
 ```yarn start```
 
Test

```yarn test```
